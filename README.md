# PythonShootGame

A simple shoot game written in Python.


# Rule
+ 적이 한 번 죽을 때 마다 점수 1000 증가
+ 2^(플레이어 레벨) 만큼의 적을 죽일 때마다 레벨업
+ 50000점 넘을 시 보스 출현
+ 떨어지는 운석에 맞으면 죽음
+ 랜덤으로 바람이 나와 player가 좌우로 이동
